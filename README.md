# GraviCLI
un gioco dove devi cambiare la gravità del tuo terminale e molto altro per andare avanti

## Build

- Su Windows usa `build.bat` o `compila.bat`.
- Su Linux e macOS usa `build.sh` o `compila.sh`.
- Su macOS il progetto segue il ramo POSIX, quindi non usa gli header/flag specifici di Windows.
